Visual Harmony

Where sound meets sight.

Visual Harmony is a clean, interactive web experience that transforms audio into visual patterns. It’s not just about playing sound—it’s about seeing it.

Whether you're experimenting with music, chilling with visuals, or just curious about how sound behaves, this app turns raw audio into something you can actually feel.

🚀 Live Demo

👉 https://visual-harmony--kprajurkar.replit.app

🧠 What It Does
Converts audio into dynamic visual patterns
Reacts in real-time to frequency + amplitude
Creates a smooth, immersive visual experience
Minimal UI so the visuals stay the star

Think of it like… a music visualizer, but stripped down to its purest form.

⚙️ Tech Stack
HTML5 Canvas – for rendering visuals
JavaScript – core logic + animation
Web Audio API – extracting sound data
CSS – simple, clean styling
✨ Features
🎵 Real-time audio visualization
🎨 Smooth animated graphics
⚡ Lightweight and fast
🧩 Simple structure (easy to tweak & build on)
🛠️ How It Works (Quick Breakdown)
Audio input is captured
Web Audio API analyzes frequencies
Data is mapped to visual parameters
Canvas renders dynamic visuals frame-by-frame

Basically:
Sound → Data → Motion → Art
